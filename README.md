# Pinterest Video Automation Engine 🚀

Автоматизированная система подготовки и публикации уникального видеоконтента для Pinterest. Скрипт берет данные из базы, создает анимированное видео из статичного изображения и оптимизирует метаданные с помощью ИИ.

## 🛠 Технологический стек
* **Python 3.x**
* **База данных:** Supabase (PostgreSQL)
* **Хранилище:** Cloudflare R2 (S3-совместимое)
* **Обработка видео:** FFmpeg (эффекты Zoom/Pan)
* **AI/LLM:** Ollama (локальный запуск модели Qwen 2.5) для генерации описаний
* **Рендеринг:** Browserless (Puppeteer) для создания сложных макетов из HTML/CSS

## 📋 Основной функционал
1. **Динамический рендеринг:** Превращает входные данные в стилизованные PNG-карточки через HTML-шаблоны.
2. **Video Processing:** Конвертирует изображения в 5-секундные MP4-ролики с эффектом плавного приближения.
3. **AI Optimization:** Автоматически создает SEO-оптимизированные заголовки, описания и Alt-тексты, используя локальную LLM.
4. **Cloud Sync:** Автоматическая загрузка готовых медиафайлов в Cloudflare R2 и обновление статусов в Supabase.
5. **Scheduler:** Встроенный планировщик для непрерывной обработки очередей задач.
